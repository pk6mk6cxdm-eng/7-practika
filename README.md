# 7-practika
Берілгендері:

Эксперимент класы: атауы, күні, нәтижесі. JSON арқылы сақтау, файлдан оқу, ең соңғы экспериментті экранға шығару.

Экологиялық бақылау есептерінде аймақ аты, су сапасы мәндері және ауаның ластануы индекстері бар. Файлға жазылады. Файлдан оқып, ең таза суды көрсететін аймақты табу, ауа ластануы 50 индекстен жоғары аймақтарды бөлу қажет.
